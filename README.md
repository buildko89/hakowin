# 箱庭関連のWindows用のインストーラとアプリケーションのリポジトリ

箱庭ラボのリポジトリにある箱庭コア機能、箱庭ドローンシミュレータなどをWindows上で利用できるようにするためのインストーラ
とアプリケーションを公開しています。

- 箱庭ラボのリポジトリ
[箱庭コア機能](https://github.com/hakoniwalab/hakoniwa-core-pro.git)
[箱庭ドローンシミュレータ](https://github.com/toppers/hakoniwa-drone-core/tree/main)


|ディレクトリ名|説明|
|:---|:---|
|hakoCoreInstaller|箱庭コア機能をWindowsにインストールするためのインストーラ|
|hakoAppsInstaller|箱庭ドローンシミュレータをWindowsにインストールするためのインストーラ|
|hakoWinAppsAPI|箱庭ドローンシミュレータのPython APIを使ったドローン操作させるアプリケーション|
|hakoWinAppsRC|箱庭ドローンシミュレータをゲームコントローラなどでドローンを操縦するアプリケーション|


利用にあたっては、以下のドキュメントを参照してください。
