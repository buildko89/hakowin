# hakoAppsInstallerの説明

箱庭ドローンシミュレータをWindowsにインストールするためのインストーラになります。

インストーラの開発は、Visual Studio Community 2022のSetup Projectを使って開発しています。
対応言語は、C#になります。


開発にあたっての詳細な説明は、以下のドキュメントを参照してください。

[Windows用のインストーラ開発方法](https://github.com/buildko89/documents/blob/main/WinInstller_DevDocuments/windows_hakowin_installer_develop.md)
